# SIMPLE-API-NODEJS
It is a demonstrative api with mongoose and express

## How to run the app

Clone the project
```
  git clone https://github.com/guisantosalves/SIMPLE-API-NODEJS.git
```

Install dependencies
```
  npm install
```

start the server
```
  npm start
```

<p>by default the server will run into port :3000</p>
